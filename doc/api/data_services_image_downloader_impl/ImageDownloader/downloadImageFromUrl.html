Downloads an image from the given url and saves it to the savePath.

Sets completed to true if the download is successful, otherwise false. Returns true if the download and saving process is successful, otherwise false.

Implementation

@override
Future<bool> downloadImageFromUrl(String url, String savePath) async {
  completed = false;
  try {
    await dio.download(
      url,
      savePath,
    );
    completed = true;
  } catch (e) {
    completed = false;
  }
  return completed;
}
