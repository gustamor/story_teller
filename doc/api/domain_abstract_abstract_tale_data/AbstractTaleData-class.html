AbstractTaleData

AbstractTaleData class abstract

Abstract class defining the structure of tale data.

This class outlines the properties and methods that any tale data representation must have. Implementations of this class are expected to provide concrete details for each of these abstract members. The class includes essential tale-related attributes like ID, title, story, and prompt.
