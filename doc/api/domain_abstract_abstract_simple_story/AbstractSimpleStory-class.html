AbstractSimpleStory

AbstractSimpleStory class abstract

Abstract class representing the basic structure of a story.

This class outlines the essential attributes of a story, including its title, associated speech document, user information, date, UUID, image URL, and text content. It provides a template for story-related operations like copying and serialization.
