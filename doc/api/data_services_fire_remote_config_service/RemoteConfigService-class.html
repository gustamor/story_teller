RemoteConfigService class

Service for managing and initializing Firebase Remote Config.

Implements AbstractFirebaseRemoteConfigService to provide functionalities for configuring and initializing Firebase Remote Config. It handles setting up default values and fetch intervals, and activates fetched configurations.
