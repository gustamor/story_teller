AbstractAiClickableCard

AbstractAiClickableCard class abstract

Represents an abstract clickable card with customizable properties.

This abstract class provides a structure for clickable cards, allowing consistent styling and behavior customization. It is designed to support platform-specific customizations for different platforms like Android and iOS.
