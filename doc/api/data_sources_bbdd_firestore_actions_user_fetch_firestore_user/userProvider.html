StreamProvider<User?> userProvider
final

Provides a stream of the current Firebase user.

This provider listens to the user's data changes in the Firebase database and emits the user data whenever it changes. If the user data is not available or an error occurs, it emits null.

Implementation

final userProvider = StreamProvider<firebaseUser.User?>((ref) async* {
  final userReference = await ref.watch(fetchCurrentUserProvider.future);
  yield* userReference.snapshots().map((snapshot) => snapshot.data());
});
